### Фигуры Лиссажу

> *В зависимости от угла Alpha и соотношения между частотами (nuX и nuY)*

Координаты задаются формулами (t пробегает от 0 до 2pi):

 ```
x = A * sin(nuX * t)
y = B * sin(nuY * t + Alpha)

 ```
